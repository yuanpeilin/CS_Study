# 表格
* **`table`** 表示表格标签
    * border 设置边框
    * cellspacing 设置单元格间距
* **`tr`** (table row)是行标签
* **`th`** (table head)是表头单元格标签
* **`td`** (table data)是普通单元格标签
    * colspan 设置跨列
    * rowspan 设置跨行
```html
<table border="1" cellspacing="0">
    <tr>
        <th>1.1</th>
        <th>1.2</th>
        <th>1.3</th>
    </tr>
    <tr>
        <td>2.1</td>
        <td>2.2</td>
        <td>2.3</td>
    </tr>
    <tr>
        <td>3.1</td>
        <td>3.2</td>
        <td>3.3</td>
    </tr>
</table>
```
