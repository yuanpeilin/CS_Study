# 新建数据库
### 语法
```sql
CREATE DATABASE [IF NOT EXISTS] <DATABASE_NAME> [ [DEFAULT] CHARACTER SET [=] <CHARACTER_SET_NAME> ];
```

### 例子
```sql
CREATE DATABASE test CHARACTER SET utf8;
CREATE DATABASE test CHARACTER SET = utf8;
CREATE DATABASE test DEFAULT CHARACTER SET utf8;
```
